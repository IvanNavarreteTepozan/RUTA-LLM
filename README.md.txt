# Mathematical Foundations of Artificial Intelligence (MAT 4953/6973)

## Project Description
This repository documents the code, mathematical derivations, and practical implementations developed for the **MAT 4953/6973: Math Found AI** course. The assignment design and mathematical requirements were guided by Professor Julio César, following the course structure by Dr. Eduardo Dueñez.

The primary objective of this repository is to bridge theoretical mathematics with applied machine learning. It covers the rigorous application of calculus, linear algebra, probability, and statistics that underpin modern AI, ranging from manual matrix decompositions to the training of deep generative models.

## Repository Structure
The project is organized into five main directories, each corresponding to a core course assignment:

* **📁 Assignment_1_Feedforward_Networks:** Baseline implementation of feedforward neural networks using Keras. Includes hyperparameter experimentation (network width, depth, and epochs) and performance analysis using the MNIST dataset.
* **📁 Assignment_2_Linear_Algebra_SVD:** Applied linear algebra focusing on the exact, manual calculation of Singular Value Decompositions (SVD). Includes numerical verification via NumPy, PCA analysis, and practical applications in lossy image compression.
* **📁 Assignment_3_Probability_Information:** Theoretical probabilistic analysis covering multivariate normal distributions, the mathematical relationship between SVD and covariance matrices, and exact closed-form derivations of Kullback-Leibler (KL) divergence for Normal and Bernoulli distributions.
* **📁 Assignment_4_Optimization_Tradeoffs:** Practical exploration of the Bias-Variance Tradeoff and machine learning optimization techniques, specifically focusing on Stochastic Gradient Descent (SGD) implementations using scikit-learn.
* **📁 Assignment_5_Advanced_Architectures:** Deep dives into complex architectures and generative modeling. Covers manual backpropagation derivations (chain rule, cross-entropy), self-attention mechanisms in Transformers, and the mathematical framework of Variational Autoencoders (VAEs), including ELBO derivations and temperature scaling.

## Technologies & Tools Utilized
* **Core Language:** Python
* **Deep Learning Frameworks:** TensorFlow & Keras
* **Machine Learning & Data Processing:** scikit-learn, NumPy
* **Documentation & Proofs:** LaTeX for rigorous mathematical typesetting and analytical reports.
